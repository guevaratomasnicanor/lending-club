# lending-club
Las variables mas discriminantes entre prestamos buenos y malos son las siguientes:
int_rate (tasa de interés)
inq_last_6mths, inq_last_12m, inq_fi (consultas crediticias)
open_il_12m, open_il_24m (nuevas líneas de crédito abiertas)
all_util, revol_util (utilización del crédito total)
-Algunos estados como AK, DC SD tienen altas tasas de malos prestamos
-Subgrupos del D5 al G5 tienen altas tasas de malos prestamos
-Los prestamos a 60 meses son levemente peores a los de 36.
-Los verificados son peores.
